Setup Instructions

Method 1 — Copy & Paste (Recommended)

  1. Fill in your details in the form above
  2. Click Copy Signature
  3. Open Outlook → File → Options → Mail → Signatures
  4. Click New and name it “Meridian Grand”
  5. Click inside the signature editing box, then press Ctrl+V to paste
  6. Click OK to save
  7. Set it as your default signature for new messages and replies
Tip: If images appear broken after pasting, try the “Download .htm” method below instead.

Method 2 — File Placement (For IT)

  1. Click Download .htm above to get the signature file
  2. Place the .htm file in: %APPDATA%\Microsoft\Signatures\
  3. Open Outlook → File → Options → Mail → Signatures
  4. Select “Meridian Grand” from the list and set as default
